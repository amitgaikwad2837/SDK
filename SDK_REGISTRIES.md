## Publishing Configuration

### npm Publishing
- **Workflow**: `.github/workflows/publish-web-sdks.yml`
- **Trigger**: Push to main + path changes OR manual dispatch
- **Auth**: `NPM_TOKEN` (automation token with 2FA bypass)
- **Scope**: All 15 web SDKs published in parallel
- **Independence**: fail-fast: false (one SDK failure doesn't block others)

### Maven Central Publishing
- **Workflow**: `.github/workflows/publish-android-sdks.yml`
- **Trigger**: Push to main + path changes OR manual dispatch
- **Auth**: CENTRAL_PORTAL_* or SONATYPE_* + SIGNING_* credentials
- **Scope**: All 8 Android SDKs published in parallel
- **Independence**: fail-fast: false (one SDK failure doesn't block others)

```bash
npm install @amitgaikwad37/api-contract-drift-detector
```

