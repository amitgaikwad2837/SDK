# Public SDK Ideas Workspace

This repository contains 15 SDK/CLI/Action project ideas organized so each can be developed independently.

## Goals

- Keep each project isolated and easy to iterate.
- Maintain consistent documentation across all ideas.
- Enable parallel development and experimentation.

## Repository Structure

Each project lives under `projects/<project-slug>/` with:

- `README.md`: Overview, value proposition, and scope.
- `docs/requirements.md`: Product and technical requirements.
- `docs/implementation-plan.md`: Milestones and execution steps.
